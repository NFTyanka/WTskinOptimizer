EU: The program is designed to optimize WT skins by dividing the height and width of the image by 2. The program is able to compress skins 2 and 4 times, as well as stretch them 2 times. The program is in the early stages of development and will be actively modified and improved, new functions will be added, etc. 

At the moment, the program is localized for the English-speaking segment of users, and also has the function of remembering the latest settings, including the language.

Installation and usage instructions:

=> Download the Optimizer v1.X.zip

=> Unzip the Optimizer-1.X folder from .zip file

=> Run "WT_Skin_Optimizer"

=> Set the language you need

=> Specify the path to the folder with your skin - (select skin folder)

=> Specify the download path of the folder with the updated skin (optional) - (change output folder)

=> Change the postfix in the folder name (optional) - (folder name suffix)

=> Click "Start Processing"

=> By default, the "Downloads" path will have your edited skin in the folder labeled "_DSCL", but you can also change the path and the name of the folder's postfix using the points above.

=> Move the updated folder to the UserSkins folder in the root of the War Thunder game

=> Upgrade the pool of skins in the game and install a new skin!


Good luck!

RU: Программа предназначена для оптимизации скинов WT путем деления на 2 значений высоты и ширины картинки. Программа способна ужимать скины в 2 и 4 раза, а так же растягивать их в 2 раза. Программа на ранних стадиях разработки и будет активно модифицироваться и улучшаться, будут добавляться новые функции и тп. 

На данный момент программа локализирована для англоязычного сегмента пользователей, а так же имеет функцию запоминания последних настроек, в том числе и языка.

Инструкция по установке и использованию:

=> Скачайте Optimizer v1.X.zip

=> Распакуйте папку Optimizer-1.X из .zip файла

=> Запустите "WT_Skin_Optimizer"

=> Установите нужный вам язык

=> Укажите путь до папки с вашим скином - (Выбрать папку скина)

=> Укажите путь загрузки папки с обновленным скином(по желанию) - (Изменить папку сохранения)

=> Измените постфикс в названии папки(по желанию) - (Приписка к названию папки)

=> Нажмите "Начать обработку"

=> По умолчанию по пути "Downloads" будет ваш отредактированный скин в папке с припиской "_DSCL", но вы так же по своему желанию можете изменить путь и название постфикса папки с помощью пунктов выше.

=> Переместите обновленную папку в папку UserSkins в корне игры War Thunder

=> Обновите пул скинов в игре и установите новый скин!
