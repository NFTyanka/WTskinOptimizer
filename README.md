EU: The program is designed to optimize WT skins by dividing the height and width of the image by 2. The program is able to compress skins 2 and 4 times, as well as stretch them 2 times. The program is in the early stages of development and will be actively modified and improved, new functions will be added, etc.

Installation Instructions:

-> Download the Optimizer 1.1.zip

-> Unzip the Optimizer-1.1 folder from .zip file

-> Run "WT_Skin_Optimizer"

-> Specify the path to the folder with your skin

-> Click "Start"

-> On the "Downloads" path, your edited skin will be in the folder labeled "_DSCL"

-> Move this folder to the UserSkins folder in the root of the War Thunder game

-> Upgrade your skin pool and install a new skin!


Good luck!

RU: Программа предназначена для оптимизации скинов WT путем деления на 2 значений высоты и ширины картинки. Программа способна ужимать скины в 2 и 4 раза, а так же растягивать их в 2 раза. Программа на ранних стадиях разработки и будет активно модифицироваться и улучшаться, будут добавляться новые функции и тп.

Инструкция по установке:
-> Скачайте Optimizer 1.1.zip
-> Распакуйте папку Optimizer-1.1 из .zip файла
-> Запустите "WT_Skin_Optimizer"
-> Укажите путь до папки с вашим скином
-> Нажмите "Пуск"
-> По пути "Downloads" будет ваш отредактированный скин в папке с припиской "_DSCL"
-> Переместите эту папку в папку UserSkins в корне игры War Thunder
-> Обновите пул скинов и установите новый скин!

Удачи!
